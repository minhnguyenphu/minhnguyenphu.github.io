CNN From Scratch — Không dùng pre-trained weights

Bệnh lý lồng ngực là nhóm bệnh có tỷ lệ tử vong cao nhất toàn cầu. Dự án xây dựng Custom CNN 4-Block Double-Conv huấn luyện hoàn toàn từ đầu trên bộ dữ liệu NIH Chest X-ray 14 — chứng minh rằng kiến trúc tự thiết kế có thể vượt baseline học thuật (Wang et al. 2017) và thu hẹp khoảng cách với CheXNet SOTA.

Câu hỏi cốt lõi: "Một CNN tự thiết kế, from scratch, có thể đạt đến đâu?" — Câu trả lời: Mean AUC 0.7802, vượt Wang et al. trên 11/14 bệnh lý.

01 / LOSS FUNCTION

Focal Loss

α = 0.65, γ = 2.0, float32-safe. Giải quyết mất cân bằng nhãn nghiêm trọng — Hernia chỉ chiếm 0.2% dataset. Thay thế BCE tiêu chuẩn để tập trung vào hard examples.

FL(p) = −α(1−p)^γ log(p)
02 / TRAINING

Progressive Training 4 Giai đoạn

Mở dần từng block — từ Block 1 đến Block 1–4. Tăng resolution từ 224×224 lên 384×384 ở Stage 4 sau khi model đã warm-up, đảm bảo hội tụ ổn định.

224px → 384px
03 / HARDWARE

Dual GPU + Mixed FP16

MirroredStrategy trên 2× NVIDIA Tesla T4. Mixed Precision FP16 giảm VRAM, tăng throughput. Global Batch Size 64 = 32/GPU.

T4 × 2 | FP16
04 / EVALUATION

TTA + Per-class Threshold

Test Time Augmentation giảm variance. Tối ưu ngưỡng riêng từng bệnh qua Youden Index. Hạ thêm 15% ngưỡng cho 4 bệnh hiếm để giảm false negative.

Youden Index | −15% Rare
05 / SAMPLING

Class Weights Động

Tính trọng số dựa trên phân phối nhãn thực tế trong Train Set. Normalize về [1.0×, 5.0×] — kết hợp Focal Loss tạo áp lực học kép trên minority classes.

Range [1.0×, 5.0×]

Kiến trúc mô hình

Custom CNN — 4 Block Double-Conv

~2.88M tham số. Thiết kế theo nguyên tắc "Sâu dần — Rộng dần — Double Conv". Không dùng pre-trained weights.

// MODEL ARCHITECTURE
Input: (B, 384, 384, 3)
│
BLOCK 1 — Low-level Features
Conv2D(32) × 2 → BN → ReLU → Pool → Drop(0.25)
→ 192×192
│
BLOCK 2 — Texture / Pattern
Conv2D(64) × 2 → BN → ReLU → Pool → Drop(0.25)
→ 96×96
│
BLOCK 3 — Semantic Features
Conv2D(128) × 2 → BN → ReLU → Pool → Drop(0.25)
→ 48×48
│
BLOCK 4 — High-level Pathology
Conv2D(256) × 2 → BN → ReLU → Pool → Drop(0.25)
→ 24×24
│
HEAD — Classifier
GlobalAvgPool2D → Dense(512) → BN → Drop(0.5)
→ Dense(15, sigmoid)
│
Output: (B, 15) — xác suất độc lập
Stage 01 — Warm-up

Block 1 + Head

Chỉ mở Block 1 và Classifier Head. Block 2–4 bị freeze. Học đặc trưng thô: cạnh, vân, ranh giới mô.

224×224 · LR=1e-4 · ~2h
Stage 02 — Expand

Block 1–2 + Head

Mở thêm Block 2. Block 3–4 còn freeze. Học texture và pattern tổn thương, kết hợp đặc trưng thô từ Stage 1.

224×224 · LR=1e-4 · ~2h
Stage 03 — Deepen

Block 1–3 + Head

Mở Block 3. Chỉ Block 4 còn freeze. Học đặc trưng ngữ nghĩa cao — hình thái và phân bố bệnh lý.

224×224 · LR=1e-4 · ~2h
Stage 04 — Full Resolution ★

Tất cả Block + Head

Unfreeze tất cả. Nâng từ 224 lên 384×384 — cung cấp pixel-level detail để phân biệt Nodule vs Mass, đặc trưng mịn của Fibrosis.

384×384 · best_stage4.keras (110 MB)
Warm Restart

Cosine Annealing

Thoát cực tiểu cục bộ, khám phá không gian tham số rộng hơn. ReduceLROnPlateau + EarlyStopping.

~30 phút
